LIVELLO 0

Cos'è un prompt

💡 Idea Chiave
  • L'obiettivo è semplice:
  • 1. Dire cosa vuoi
  • 2. Dirlo in modo chiaro
  • 3. Ottenere una risposta utile
🎯 Esempi pratici
🎯 Esempio semplice
Prompt poco chiaro: "Parlami di cucina"
🎯 Esempio:
"Spiega come preparare il caffè con la moka, passo per passo". Cosa aspettarsi:
✅ Consigli Pratici
  • Se è confusa, il prompt va migliorato.
  • • La risposta è confusa → riscrivi il prompt più preciso
  • Stai già imparando una cosa importante, fare richieste chiare è una grande abilità, andiamo avanti con calma.
